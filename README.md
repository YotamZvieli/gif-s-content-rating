#Gifs content rating project
##Application
###Install & Run tha app:
- From `gits-rating-app/frontend` run:
```shell
npm install
npm start
```
- From ```gits-rating-app/Python-Backend``` run:
```shell
pip install open_clip_torch
pip install numpy
python gift-rate-server.py
```
 Open the app on [App](http://localhost:3000)
